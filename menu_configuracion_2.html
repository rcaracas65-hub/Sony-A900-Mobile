11. Menú Configuración 2

Administración de tarjetas de memoria, numeración y organización de carpetas, así como la configuración de la conexión USB para transferencias a computadora.

11.1 Tarjeta memoria

Selecciona qué ranura utilizar: Cf (CompactFlash) o MS (Memory Stick Duo). Puedes trabajar con dos tarjetas y cambiar rápidamente entre ellas.

[Pág. 124 – Tarjeta memoria]

11.2 Número archivo

Define cómo se numeran los archivos:
• Serie – La numeración continúa aunque cambies o formatees la tarjeta.
• Restablecer – Reinicia la numeración desde 0001 al crear una carpeta nueva o formatear.

[Pág. 125 – Número archivo]

11.3 Nombre carpeta

Permite elegir el tipo de nombre para las carpetas donde se guardan las imágenes: • Estándar – Formato DSCxxxx.
• Fecha – Crea carpetas según la fecha (ej.: 101MSDCF).

[Pág. 125 – Nombre carpeta]

11.4 Selecc.carpeta / Carpeta nueva

Permite gestionar las carpetas en la tarjeta de memoria:
• Seleccionar carpeta – Elige la carpeta donde quieres que se guarden las imágenes.
• Crear carpeta nueva – Útil para separar sesiones, viajes o proyectos.

[Pág. 125 – Selecc.carpeta / Carpeta nueva]

11.5 Conexión USB

Define cómo se comporta la cámara al conectarla por USB a la computadora:
• Control PTP – Protocolo de transferencia de imágenes (compatible con software específico).
• Almacenamiento masivo – La cámara aparece como unidad externa (recomendado).

[Pág. 126 – Conexión USB]

11.6 Tarj.AlmMasivo

Configura el modo de la cámara para utilizarse como dispositivo de almacenamiento masivo USB. Permite copiar imágenes directamente sin software adicional.

[Pág. 126 – Tarj.AlmMasivo]
