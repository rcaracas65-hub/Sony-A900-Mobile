13. Menú Configuración 4

Ajustes relacionados con el tamaño de captura APS-C y la reposición general de la cámara a los valores predeterminados.

13.1 Tam.capt. APS-C

Permite capturar imágenes utilizando solo un área central del sensor equivalente al formato APS-C en lugar de usar todo el fotograma completo (full frame). En el visor se muestra un marco que indica el área APS-C.

• Activar – La cámara toma la foto usando el área de tamaño APS-C (recorte).
• Desactivar – Se utiliza toda el área del sensor (tamaño normal de la A900).

Esto puede ser útil si quieres “acercar” un poco más sin cambiar de objetivo, o para mantener coherencia con otras cámaras APS-C en tu flujo de trabajo.

Referencia manual: [Pág. 130 – Tam.capt. APS-C]

13.2 Reponer predet. (Reset general)

Restaura una gran cantidad de ajustes de la cámara a sus valores predeterminados de fábrica. Afecta menús de grabación, reproducción y configuración.

Entre los elementos que se reponen están, por ejemplo:

  • Visualización de info de grabación → vista detallada estándar.
  • Compensación de exposición → ±0.0.
  • Modo de manejo (Drive) → Avance sencillo.
  • Balance de blancos → AWB, temperatura base alrededor de 5500K.
  • ISO → AUTO.
  • Visualización de reproducción → pantalla de imagen única con datos.
  • Tamaño imagen → L: 24M.
  • Propor.dim. → 3:2.
  • Calidad → JPEG Fina.
  • Optim.gama diná (DRO) → Desactivado.
  • BotónPersonaliz. → Estilo.Creativo.
  • Estilo Creativo → Estándar.
  • Rango color → sRGB.
  • Diapositivas – Intervalo → 3 s.
  • Brillo LCD → 0.
  • Tiem.visu.info. → 5 s.
  • Ahorro energía → 3 min.
  • Salida HDMI → Prioridad 1080i.
  • Número archivo → Serie.
  • Nombre carpeta → Forma normal.
  • Conexión USB → Almacenamiento masivo.
  • Inicio menú → Principal.
  • Confirm.borrar → “Cancelar” como opción por defecto.
  • Señales audio → Activar.
  • Micro ajuste AF → Desactivar.
  • Tam.capt. APS-C → Desactivar.

Es una opción útil cuando has cambiado muchos parámetros y quieres regresar la cámara a una base consistente sin tener que ir opción por opción.

Referencia manual (lista completa de elementos que se reponen): [Pág. 130 – Reponer predet. (inicio)], [Pág. 132 – Elementos reponer a ajuste predeterminado]
