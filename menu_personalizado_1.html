5. Menú Personalizado 1

Configura el comportamiento del enfoque automático, selección AF, velocidad AF y prioridad del disparo en AF-S o AF-C.

5.1 Botón AF/MF

Cambia de AF/MF mientras se mantiene apretado el botón AF/MF
• Control AF/MF → El botón se utiliza para cambiar entre enfoque automático y enfoque manual
• Bloqueo AF → El botón se utiliza como botón de bloqueo AF. Pulse el botón disparador para tomar imágenes mientras mantiene pulsado el botón AF/MF.

Referencia: [Pág. 111 –Cambio sencillo de AF/MF]

Ver Detalle: [Pág. 57 –Cambio sencillo de AF/MF]

5.2 Control AF/MF

Si [Botón AF/MF] se ajusta en [Control AF/MF], se puede cambiar el modo de operación del botón AF/MF.
• Retener → AF y MF se conmutan sólo mientras se mantiene pulsado el botón.
• Alternar → Pulse y suelte el botón para cambiar entre AF y MF.

Referencia: [Pág. 111 –Control AF/MF]

5.3 Velocidad AF

Puede seleccionar velocidad de enfoque cuando se activa el enfoque automático.
• Rápido → Confirma el enfoque rápidamente.
• Lento → Confirma el enfoque despacio.

[Pág. 111 – Velocidad AF]

5.4 Visual.área AF

Cuando se ajusta el enfoque, el área AF local enfocada se ilumina brevemente en rojo.
• 0,6 s → El área AF local se ilumina durante 0,6 segundos.
• 0,3 s → El área AF local se ilumina durante 0,3 segundos.
• Desactivar → El área AF local se ilumina sólo cuando se pulsa el multiselector.

[Pág. 111 – Visual.área AF ]

5.5 Botón ret.enfoq

Si utiliza un objetivo equipado con un botón de retención del enfoque, puede cambiar la función del botón para previsualizar una toma.
• Ret.enfoque → El botón se utiliza como botón de retención del enfoque.
• Previsual.Ópti → El botón se utiliza como botón de previsualización óptica.
• Previs. Intel. → El botón se utiliza como botón de previsualización inteligente.

[Pág. 112 – Botón ret.enfoq]

5.6 Rev.automática

Después de grabar una imagen, puede visualizarla en el monitor LCD. Puede cambiar el tiempo de visualización.
• 10 s, 5 s, 2 s, → Muestra la imagen en el monitor LCD durante el tiempo seleccionado.
• Desactivar → Desactiva la revisión automática. inteligente.
• Nota → En la revisión automática no se visualizará la imagen en posición vertical aunque haya tomado la imagen en posición vertical.

[Pág. 120 – Rev.automática]

[Pág. 112 – Rev.automática]

5.7 Func.Previsual.

Puede seleccionar la función de previsualización. Para ver detalles sobre la función de previsualización,consulte la página:

[Pág. 84 – Utilización de la función de previsualización]
• Inteligente → Utiliza la función de previsualización inteligente
• 0,3 s → El área AF local se ilumina durante 0,3 segundos.
• Previsual.Ópti → Utiliza sólo la función de previsualización óptica.
