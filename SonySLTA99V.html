Menú de toma de imágenes fijas

Referencia manual: Abrir PDF en la página 35 (Menú de toma de imágenes fijas)

Ajustes principales para fotos fijas: tamaño, calidad, relación de aspecto, funciones de reducción de ruido, flash, estabilización y comportamiento del disparador.

Elemento Descripción Opciones principales
Menú 1 (Toma de imágenes fijas)
Tamaño de imagen Selecciona el tamaño de las imágenes fijas. L / M / S
Relación de aspecto Define la proporción del fotograma para las fotos. 3:2 / 16:9
Calidad Define la calidad de las imágenes fijas. RAW / RAW & JPEG / Extrafina / Fina / Estándar
Panorama: tamaño Selecciona el tamaño de las imágenes panorámicas. Estándar / Ancho
Panorama: dirección Define la dirección de barrido para las imágenes panorámicas. Derecha / Izquier. / Arriba / Abajo
Menú 2 (Toma de imágenes fijas)
Zoom Imag. Clara Acerca la imagen con un zoom de mejor calidad que el zoom digital convencional. Activar / Desactivar
Zoom digital Acerca la imagen con un zoom de ampliación mayor, sacrificando algo de calidad. Activar / Desactivar
RR Exp. Larga Reducción de ruido para exposiciones largas (aprox. 1 segundo o más). Activar / Desactivar
RR ISO Alta Reducción de ruido al fotografiar con sensibilidades ISO altas. Normal / Bajo / Desactivar
Control flash Método de medición para determinar la cantidad de luz del flash. Flash ADI / Preflash TTL
Iluminador AF Activa la luz de ayuda al enfoque para motivos poco iluminados. Automático / Desactivar
Menú 3 (Toma de imágenes fijas)
SteadyShot (foto) Estabilización de imagen para reducir trepidación al hacer fotos. Activar / Desactivar
Velocidad AF Ajusta la velocidad de enfoque automático (útil por ejemplo en macro o motivos estáticos). Rápido / Lento
Rango color Selecciona el espacio de color utilizado para las imágenes fijas. sRGB / AdobeRGB
Controlad. silencioso Asigna funciones al multicontrolador silencioso para modificar parámetros (por ejemplo ISO, compensación de exposición, modo de medición, etc.) durante la toma de fotos sin hacer ruido. Función asignable (ISO, Comp. exp., Medición, etc.)
Menú 4 (Toma de imágenes fijas)
Paso exposición Elige el tamaño del incremento en pasos de exposición para velocidad, abertura y compensación. 0,5 EV / 0,3 EV
Ajust.prioridad Define si el obturador puede disparar aunque el enfoque no esté confirmado en AF, o si se prioriza el enfoque. AF / Liberar
AF c/obturador Determina si el enfoque automático se activa al pulsar el disparador hasta la mitad, permitiendo separar AF y medición/exposición. Activar / Desactivar

Menú de grabación de películas

Referencia manual: Abrir PDF en la página 37 (Menú de grabación de películas)

Ajustes específicos para video: formato, tamaño, estabilización, seguimiento AF, sonido y reducción de ruido de viento.

Elemento Descripción Opciones principales
Menú 1 (Grabación de películas)
Formato de archivo Selecciona el formato en el que se grabarán las películas. AVCHD / MP4
Tamaño de imagen (Película) Define la resolución y tasa de bits de las películas. 60i/50i 24M(FX), 17M(FH)
60p/50p 28M(PS)
24p/25p 24M(FX), 17M(FH)
1440×1080 12M / VGA 3M
SteadyShot (Película) Estabilización de imagen para grabación de películas. Activar / Desactivar
Durac. seguimiento AF Sensibilidad del seguimiento del enfoque automático durante la grabación de video. Alto / Medio / Bajo
Auto obturador lento Permite bajar la velocidad de obturación automáticamente en escenas oscuras para vídeo. Activar / Desactivar
Controlad. silencioso (Película) Asigna funciones al multicontrolador silencioso para realizar ajustes sin ruido durante la grabación. Nivel audio / Modo AF / Área AF / ISO / Comp. exp. / Medición / Vel. obturador / Abertura
Menú 2 (Grabación de películas)
Grabación de sonido Activa o desactiva la grabación de audio. Activar / Desactivar
Reducc. ruido viento Reduce el ruido del viento durante la grabación de audio. Activar / Desactivar
Visual. nivel aud. Muestra u oculta el medidor del nivel de audio. Activar / Desactivar
Nivel grabac. audio Ajusta la sensibilidad del micrófono al grabar vídeo. 0 a 31
Tiempo salida audio Ajusta la latencia del audio en los auriculares para monitoreo. Útil para reducir eco o desfase. En vivo / Sincronización labios
