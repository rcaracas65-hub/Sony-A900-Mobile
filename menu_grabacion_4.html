4. Menú Grabación 4

Menú orientado a registrar tus propios ajustes en memoria y a reponer las funciones principales del modo de grabación a sus valores predeterminados.

4.1 Memoria

Permite registrar hasta tres combinaciones de ajustes de uso frecuente (modos de exposición, modo de manejo, ISO, balance de blancos, modos de enfoque, área AF local, etc.) y asignarlas a las posiciones 1, 2 y 3 del dial de modo.

Flujo típico:

  • Pones la cámara en la configuración que quieres guardar (P/A/S/M, ISO, WB, AF, etc.).
  • Entras a [Memoria] en el Menú Grabación 4.
  • Eliges el número (1, 2 o 3) y confirmas para registrar esos ajustes.

Para recuperar los ajustes guardados, simplemente ajustas el dial de modo en 1, 2 o 3. La cámara se configura automáticamente con esa “escena personalizada”.

Referencia manual: [Pág. 86 – Registro de los propios ajustes / Memoria]

4.2 ReponerModoGrab (Reponer modo de grabación)

Restaura las funciones principales del modo de grabación a sus valores de fábrica cuando el dial de modo está en P, A, S o M. Es útil si has tocado muchos ajustes y quieres regresar a una base “limpia”.

Algunos de los elementos que se reponen:

  • Compensar exposición → ±0.0
  • Modo de manejo (Drive) → Avance sencillo
  • Balance de blancos (WB) → AWB / 5500K (según ajuste)
  • ISO → AUTO
  • Tamaño imagen → L: 24M
  • Propor.dim. → 3:2
  • Calidad → JPEG Fina
  • Optim.gama diná (DRO) → Desactivado
  • BotónPersonaliz. → Estilo.Creativo
  • Estilo.Creativo → Estándar
  • Rango color → sRGB
  • Modo flash → Flash de relleno
  • Control flash → ADI
  • Compensación flash → ±0.0
  • Paso exposición → 0,3 EV
  • Rango ISO Auto → 200–800
  • Ajustes AF-A → AF-A
  • Área AF → Ancho
  • Ajust.prioridad → AF
  • Iluminador AF → Automático
